# AI-Powered-Smart-Exam-Preparation-Platform
End-to-end UI/UX case study including web, dashboard, and mobile app design for an AI-powered exam preparation platform with personalized learning, analytics, and smart insights.
Designing an AI-powered learning experience that helps students prepare smarter, not harder.

This project showcases the UI/UX design of an AI-powered smart exam preparation platform.
The platform analyzes student performance, identifies strengths and weaknesses, and provides personalized study recommendations to improve results. By combining smart analytics with a clean and intuitive user interface, the design focuses on reducing study stress while increasing productivity and motivation.
This project demonstrates how artificial intelligence can transform traditional exam preparation into a smarter, more personalized, and more engaging learning experience.
It includes the design of a website, dashboard, and mobile application, along with user flows,
wireframes, branding, and interactive prototypes created using modern design tools.

This repository includes:
- Website UI design
- Dashboard UI design
- Mobile app UI design
- User flows and wireframes
- Branding assets
- Interactive prototypes
- Demo video

Tools used in this project:
- Figma

- Miro
- Adobe Photoshop
